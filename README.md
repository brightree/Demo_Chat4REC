# 삼성전자 Sales Agentic 챗봇

삼성전자 영업사원을 위한 AI 챗봇입니다.  
제품 정보, A/S 정책, 세일즈 전략, 맞춤 강의 추천까지 한 번에 제공합니다.

---

## 주요 기능

- 최신 제품 정보 및 Q&A (PDF 기반 RAG)
- 상황별 맞춤 강의 추천 (3~5개)
- 직관적 챗봇 UI (삼성 스타일, 퀵 리플라이)
- 대화 이력 저장 (Supabase 연동)
